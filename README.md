# Phoenix Forge Press

Phoenix Forge Press is a Next.js website starter for an independent publishing
brand focused on fantasy storytelling.

## Stack

- Next.js 16 with the App Router
- TypeScript
- Tailwind CSS v4

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

Start editing the site in `src/app/page.tsx`. Global styles and theme tokens
live in `src/app/globals.css`.

## Scripts

- `npm run dev` - start the local dev server
- `npm run build` - create a production build
- `npm run start` - serve the production build
- `npm run lint` - run ESLint

## Next Steps

- Add pages for books, authors, about, and contact
- Replace the placeholder copy with your brand voice
- Add deployment settings for Vercel or your preferred host
